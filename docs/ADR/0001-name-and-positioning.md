# ADR-0001: Name and positioning

## Status

Accepted

## Decision

Product is named Cairn.

Primary line is “Passwords, recovery, and control in one local vault.”

## Rationale

Cairn suggests a marker, safe navigation, orientation, and a way back.

The name supports recovery confidence and local control better than the previous working name VaultGuide.

## Consequences

Use `cairn` for CLI command, crates, docs, and file extension.

Historical references to VaultGuide should be removed except this ADR.
